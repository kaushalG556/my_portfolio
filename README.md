# 🌐 Portfolio Website

## 📌 Description

This is a dynamic personal portfolio website built using Django.
It showcases my skills, projects, work experience, and certifications, along with a contact system.


## 🔧 Tech Stack

* Python
* Django
* HTML, CSS, JavaScript
* MySQL / SQLite
* Bootstrap

---

## ✨ Features

### 🧠 Skills Section

* Add and manage skills with proficiency level (0–100)

### 📂 Projects Section

* Project title, description, technologies used
* Project image and live link support

### 💼 Experience Section

* Company, role, duration, and description
* Certificate upload support

### 📜 Certifications

* Certification title, platform, duration
* Certificate image and optional link

### 📩 Contact System

* User can send messages via contact form
* Messages stored in database with timestamp

---

## 📂 Project Setup

```bash
git clone https://github.com/yourusername/my_portfolio.git
cd my_portfolio
pip install -r requirements.txt
python manage.py runserver
```

---

## 🗄️ Models Overview

* **Skill** – stores skill name and proficiency
* **Project** – stores project details and links
* **Experience** – stores work experience and certificates
* **Certification** – stores course/certification details
* **ContactMessage** – stores user messages
